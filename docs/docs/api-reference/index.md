# API Reference

Welcome to the API reference for `iragca`. This section documents the public interfaces of all modules in the library.

## Modules

- [Functional](functional.md) - Function composition and pipeline utilities for building composable data transformations.

- [Matplotlib](matplotlib.md) - Professional visualization styles and accessible color palettes inspired by color vision deficiency research.
- [ML](run-logger.md) - Machine learning utilities including a lightweight metric logger for tracking training runs.

- [Warnings](warnings.md) - Deprecation management tools for maintaining clean, user-friendly APIs.
